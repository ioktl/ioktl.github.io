About

This blog is a collection of personal notes on various programming and tech related topics. It's mostly for my own benefit but if you find something interesting among the ramblings I'm glad to have shared.

I worked in system programming, graphics, and computational graph theory. My main language was C. Nowadays I like experimenting in various other fields and code in Python.

I find it useful to keep up a .plan like some other more capable minds used to do. It helps me to structure my goals and reflect on what I’m currently working on. So if you're interested in what I'm up to, you are welcome to read it. I also like to produce more thoughtful pieces on various topics which you may find under the general category of musings.
